Loan Default Detection 📉💰
This project uses machine learning to predict the likelihood of a borrower defaulting on a loan. It is designed to assist financial institutions in making data-driven lending decisions.

🔍 Project Overview
Loan default prediction is a classification problem where the goal is to identify whether a customer will repay a loan or default. This project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, evaluation, and deployment-ready code.

🧠 Models Used
Logistic Regression

Random Forest

XGBoost

Gradient Boosting Classifier

Evaluation metrics:

Accuracy

Precision

Recall

F1 Score

ROC-AUC
